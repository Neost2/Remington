# CarePath Validation Pre-Pack

This lightweight pre-pack helps you capture outreach conversations consistently and prepare for a Donna Harris submission once you reach at least 10 interviews.

## Files

- `conversation-tracker.csv`: One row per conversation.
- `signal-rubric.md`: Standard scoring (Weak/Medium/Strong).
- `synthesis-template.md`: Quick summary template to update after each interview.
- `synthesis-log.md`: Running cross-interview synthesis notes.
- `case-studies/`: Structured case-study writeups by conversation.

## How To Use

1. Add one new row to `conversation-tracker.csv` after each conversation.
2. Assign a signal score using `signal-rubric.md`.
3. Fill in one short section in `synthesis-template.md` (date + key takeaways).
4. At 10 interviews, use the tracker and synthesis notes to generate your submission pack.

## Minimum Coverage Goal (Before Submission Pack)

- At least 10 completed interviews
- Segment diversity across patients, caregivers/families, and transportation stakeholders
- At least 3 repeating pain points across segments
- Quote-level evidence plus impact details in each case
